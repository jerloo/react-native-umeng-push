### 开发环境

- nodejs 14 lts

### 开发步骤

1. 安装依赖包

```console
yarn install
```

2. 连接安卓手机或打开安卓模拟器后

```console
npx react-native run-android
```

2. 连接iOS手机或打开iOS模拟器后

```console
npx react-native run-ios
```
